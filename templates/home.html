About the Tool

This is a web tool designed to assist in the visualization of data gathered from

the paper, The History of Prestellar Cores I: Initial Density and Velocity distributions.

Select which simulation to view from, and then you will be taken to a table builder where

you can select and customize the data. Once selected, all data will be displayed for the

given cores, with an 'X' for any data that wasn't gathered.
